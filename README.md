To run this project, simply clone the repository at:
`git@github.com:GriffinLev039/lings-cars-website.git`
and then open the `index.html` file. 

If in a ZIP file, you simply need to unzip it and run the `index.html` file. 

If images are not loading/placeholders are shown, its reccommended to use the 'Live Server Extension' from VSCode to run the site.
